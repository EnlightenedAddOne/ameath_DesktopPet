## ✨ 特色功能

### 🤖 智能行为系统

- **随机游荡** - 宠物会在屏幕上自由漫步，探索各个角落
- **跟随鼠标** - 当你移动鼠标时，宠物会好奇地跟过来
- **自动休息** - 走累了会停下来休息，休息够了继续活动
- **边缘冲撞** - 偶尔会冲向屏幕边缘，触发有趣的出屏/反弹效果

### 🎨 精美动画

- 多套 idle 动画（5种不同的待机动作）
- 流畅的移动动画，支持左右方向自动切换
- 暂停/继续功能，随心控制

### 🎛️ 灵活设置

- **5档缩放** - 0.3x ~ 0.7x，随心调整宠物大小
- **鼠标穿透** - 让宠物不遮挡你的操作
- **开机自启** - 开机自动运行，陪伴你每一天
- **托盘图标** - 最小化到系统托盘，不占任务栏

### 🔧 技术特点

- **分辨率自适应** - 自动适配任意分辨率屏幕
- **轻量级** - 基于 Python + Tkinter，资源占用极低
- **单文件分发** - PyInstaller 打包，开箱即用

## 🎮 使用说明

### 基础操作

| 操作 | 说明 |
|------|------|
| 双击运行 | 启动桌面宠物 |
| 右键托盘图标 | 打开功能菜单 |
| 隐藏/显示 | 切换宠物显示状态 |
| 暂停/继续 | 让宠物停止或继续活动 |

### 托盘菜单功能

- 🖼️ **隐藏/显示** - 隐藏或显示宠物窗口
- ⏸️ **暂停/继续** - 暂停或恢复宠物活动
- 🐭 **跟随鼠标** - 开启后宠物会跟随鼠标移动
- 👆 **鼠标穿透** - 开启后鼠标可以穿透宠物，不遮挡操作
- 🔄 **开机自启** - 开机自动运行
- 📐 **缩放** - 调整宠物大小（0.3x ~ 0.7x）
- 🚪 **退出** - 退出程序

## 📋 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.12+ （源码运行需要）
- **依赖**: Pillow, pystray

## 🛠️ 技术栈

- **语言**: Python 3.12
- **GUI**: Tkinter + Pillow
- **打包**: PyInstaller
- **系统托盘**: pystray
